% KEY RULES for LuaLaTeX + polyglossia (luabidi):
%   1. \chapter{\texthebrew{...}} — fixes character reversal in title + TOC entry
%   2. \hebrewheadingformat → right-aligns chapter and HEBREW section headings
%   3. \defaultheadingformat → restores LTR BEFORE the English section 5.4
%   4. Body paragraphs in \begin{hebrew}...\end{hebrew}
%   5. English terms inside Hebrew blocks MUST use \foreignlanguage{english}{...}
%      otherwise luabidi reverses individual characters (LuaLaTeX → XeTaLauL)
%   6. \addtocontents{toc}{...} injects Hebrew TOC styles (number left, dots, title right)

% Inject Hebrew TOC style into .toc file — takes effect during \tableofcontents (pass 2+)
% Layout for Hebrew entries: number [1em] [dotfill] [1em] title [1em] page
\addtocontents{toc}{%
  \protect\etocsetstyle{chapter}{}{}%
  {\protect\addvspace{1.0em plus 1pt}%
   \begingroup\parindent0pt\noindent\bfseries%
     \etocpage\hfill{\color{uohblue}\etocname\hspace{1em}\etocnumber}%
   \par\endgroup}%
  {}%
  \protect\etocsetstyle{section}{}{}%
  {\begingroup\parindent0pt\noindent\normalfont\normalsize%
     \etocpage\hspace{0.5em}\protect\hebrewdotleaders\hspace{0.5em}%
     \etocname\hspace{0.5em}\etocnumber%
   \par\endgroup}%
  {}%
}

\section[\texthebrew{מבוא לתיאום רב-לשוני}]{\texthebrew{מבוא לתיאום רב-לשוני}\mbox{\setLTR\enspace\thesection}}

\begin{hebrew}
מערכות רב-סוכניות מודרניות נדרשות לתמוך בשפות מרובות, ובמיוחד בטקסט דו-כיווני
(\foreignlanguage{english}{BiDi}).
שפת עברית נכתבת מימין לשמאל (\foreignlanguage{english}{RTL}), בעוד שמונחים
טכניים באנגלית מוטמעים בתוך הטקסט בכיוון שמאל-לימין (\foreignlanguage{english}{LTR}).
מנוע \foreignlanguage{english}{\texttt{LuaLaTeX}} יחד עם חבילת
\foreignlanguage{english}{\texttt{polyglossia}} מספקים פתרון מלא לבעיה זו.
\end{hebrew}

\section[\texthebrew{אדריכלות מערכת הסוכנים}]{\texthebrew{אדריכלות מערכת הסוכנים}\mbox{\setLTR\enspace\thesection}}

\begin{hebrew}
הצוות שיצר מאמר זה מורכב מארבעה סוכנים המסודרים בצינור עיבוד רציף:
\begin{enumerate}
  \item \textbf{סוכן המחקר}
        (\foreignlanguage{english}{ResearcherAgent}) —
        אוסף מידע ממקורות אינטרנטיים באמצעות כלי
        \foreignlanguage{english}{DuckDuckGo}
        ומייצר קובץ הערות מחקר מובנה.
  \item \textbf{סוכן הכתיבה}
        (\foreignlanguage{english}{WriterAgent}) —
        ממיר את הערות המחקר לפרקים מובנים בפורמט
        \foreignlanguage{english}{Markdown},
        כולל פרק זה הכתוב בעברית.
  \item \textbf{סוכן העריכה}
        (\foreignlanguage{english}{EditorAgent}) —
        עורך ומשכלל את הפרקים, מוודא עקביות במינוח ובציטוטים.
  \item \textbf{סוכן ה-\foreignlanguage{english}{LaTeX}}
        (\foreignlanguage{english}{LaTeXAgent}) —
        ממיר את הפרקים הערוכים לפורמט
        \foreignlanguage{english}{LaTeX}
        ומריץ ארבעה מעברי קומפילציה עם
        \foreignlanguage{english}{\texttt{biber}}.
\end{enumerate}
\end{hebrew}

\section{Mixed-Language Example}

The following sentence demonstrates inline mixing — an English sentence containing
embedded Hebrew technical terms rendered correctly under
\texttt{polyglossia}:

\begin{hebrew}
הסוכן מבצע \foreignlanguage{english}{\emph{ReAct loop}} —
לולאת \foreignlanguage{english}{Reason-Act} שבה הסוכן מנתח,
בוחר כלי (\foreignlanguage{english}{\emph{tool}}),
מבצע, ומשנה את מצבו בהתאם לתצפית
(\foreignlanguage{english}{\emph{observation}}).
\end{hebrew}

This bilingual capability is critical for Israeli academic submissions, where citations,
technical terminology, and abstracts often mix Hebrew and English~\parencite{Segal2026}.
